# React Native Learning

## Create project with Expo

Simple JS app:
```
$ npx create-expo-app AppName
$ cd AppName
$ npx expo install react-dom react-native-web @expo/webpack-config
$ npx expo start
```

Simple TS app:
```
$ npx create-expo-app -t expo-template-blank-typescript AppName
$ cd AppName
$ npx expo install react-dom react-native-web @expo/webpack-config
$ npx expo start
```
